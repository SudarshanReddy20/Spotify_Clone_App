Spotify Clone (Demo)

Short: UI-first music demo that mimics basic Spotify playback and responsive layout.

Demo

Tech stack

Backend: Django REST (sample audio endpoints)

Frontend: React, Tailwind CSS

Audio: HTML5 Audio API (play/pause/seek)

Deployment: GitHub Pages / Vercel (frontend)

Run locally
# Serve sample backend
cd backend
pip install -r requirements.txt
python manage.py runserver

# Frontend
cd frontend
npm install
npm run dev
